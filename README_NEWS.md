# 实时新闻聚合器

一个功能完整的实时新闻聚合网页,专门为国内用户优化,支持多个国内主流新闻源。

## ✨ 功能特性

### 核心功能
- ✅ **实时新闻抓取**: 通过RSS源自动获取最新新闻
- ✅ **国内新闻源**: 支持央视新闻、新华网、人民网、凤凰网、36氪、IT之家等国内媒体
- ✅ **自动刷新**: 可自定义自动刷新间隔(30秒-5分钟)
- ✅ **搜索功能**: 实时搜索新闻标题、描述和来源
- ✅ **分类筛选**: 按科技、综合、财经、时政等分类筛选
- ✅ **响应式设计**: 完美适配桌面和移动设备
- ✅ **详情页面**: 点击查看新闻详情,支持分享和复制链接

### 界面特性
- 🎨 现代化深色主题设计
- 📱 完全响应式布局
- 🔄 平滑的动画效果
- ⚙️ 可自定义的设置面板
- 📊 实时统计信息展示
- 📄 首页简洁列表(无图片,加载更快)

## 🎯 使用方法

### 基本使用
1. 打开 `news.html`,页面会自动加载最新新闻
2. 使用顶部工具栏进行操作:
   - 🔄 点击"刷新新闻"手动更新
   - ⚙️ 点击"设置"自定义新闻源和刷新间隔

### 阅读新闻
- **首页列表**: 浏览新闻标题和摘要
- **点击详情**: 点击"查看详情"进入新闻详情页
- **阅读原文**: 在详情页点击"阅读原文"跳转到原始网站

### 搜索和筛选
- **搜索**: 在搜索框中输入关键词实时过滤
- **分类**: 点击分类按钮筛选特定类别新闻

### 自定义设置
1. 点击"设置"按钮
2. 勾选/取消勾选想要订阅的新闻源
3. 调整刷新间隔和每页显示数量
4. 点击"保存设置"应用更改

## 🔧 技术实现

### 新闻源
使用RSS2JSON API将RSS源转换为JSON格式:

```
央视新闻: https://news.cctv.com/rss/news.xml
新华网: http://www.xinhuanet.com/politics/news_politics.xml
36氪: https://36kr.com/feed
IT之家: https://www.ithome.com/rss/
```

### 核心技术
- **Fetch API**: 异步获取新闻数据
- **LocalStorage**: 保存用户设置
- **CSS Grid**: 响应式卡片布局
- **CSS Variables**: 主题色彩管理
- **Debounce**: 搜索输入防抖优化
- **URL Parameters**: 新闻详情页路由

## ⚠️ 注意事项

### CORS 问题
由于浏览器的同源策略(CORS),项目使用 rss2json.com 提供的免费转换服务来访问 RSS。

### 图片加载
- 首页不显示图片,提高加载速度
- 详情页通过跳转到原文查看完整内容

### RSS 源可用性
- 部分 RSS 源可能因政策变化而失效
- 如果某个源无法访问,可以在设置中禁用

## 🛠️ 自定义配置

### 添加新的新闻源
在 `news.js` 的 `NEWS_SOURCES` 对象中添加:

```javascript
{
    yourSource: {
        name: '源名称',
        rss: 'https://source.com/rss.xml',
        category: 'category'
    }
}
```

然后在 `news.html` 的设置弹窗中添加对应的选项。

### 修改主题颜色
在 `news.css` 的 `:root` 中修改 CSS 变量:

```css
:root {
    --primary-color: #2563eb;
    --bg-color: #0f172a;
    /* 更多颜色... */
}
```
